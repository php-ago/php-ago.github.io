What is Ago?
Upgrade to 4.x
🚀

No Dependencies

Ago is a standalone library that doesn't depend on any other libraries or extensions, making it lightweight and efficient. It's optimized for performance and ensures fast and accurate conversions.

🌍

Support for Multiple Languages

🇬🇧 🇷🇺 🇺🇦 🇳🇱 🇩🇪 🇨🇳
You can output date in several different languages such as English, Russian, Chinese, Ukrainian, Dutch, German and you can easily contribute your own language

🧩

Flexible Date and Time Parsing

Ago has a flexible date parsing and can parse several different date formats, such as UNIX timestamp, date strings, `Carbon` instances and much more

⚙️

Configurable Options to Suit Diverse Use Cases

You can configure Ago to suit your needs to the point where you can even modify the output format of the date, overwrite translation sets and more

🧪

Comprehensive Testing for Reliability

You cannot find datetime PHP library that is tested better than this one. Ago has a comprehensive test suite that covers testing for all the supported languages

📖

Thorough Documentation for Ease of Use

Ago has a detailed documentation that covers all the features of the library including older versions, so you can easily integrate it into your project
